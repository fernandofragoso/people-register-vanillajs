# people-register-vanillajs
>ES6 + CSS3 + HTML5

LocalStorage based people repository

### Requirements

1. Install [yarn](https://yarnpkg.com/en/docs/install)

2. Install [sass](https://sass-lang.com/install)

### Instructions

1. Clone this repository.

2. Install dependencies:

```shell
yarn
```

3. Run dev server:

```shell
yarn server
```

### Tests

1. Run tests

```shell
yarn test
```
